# Contribuer

## Workflow

- Travaille depuis une branche dediee.
- Garde les changements limites a un sujet clair.
- Ecris des messages de commit simples et explicites.

## Avant une pull request

- Verifie que les fichiers inutiles ne sont pas versionnes.
- Ajoute ou mets a jour la documentation si necessaire.
- Lance les verifications du projet une fois la stack en place.

## Relecture

- Decris l'objectif du changement.
- Mentionne les impacts visibles et les points a verifier.
