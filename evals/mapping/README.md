# Evals mapping

## But
Mesurer la qualité réelle des suggestions de mapping.

## Jeux de données
- cas simples
- cas ambigus
- cas historiques connus
- cas de rejet attendu

## Métriques
- exact match sur rubrique
- taux accept / reject / modify
- présence et qualité des preuves
- latence
- coût estimé
