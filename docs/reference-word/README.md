# Reference Word Documents

Ce dossier contient les documents Word de reference du projet.

## Role

- references documentaires stables
- snapshots de communication ou d'alignement externe
- support de realignement documentaire explicite quand une tache le demande

## Regles

- ces fichiers ne remplacent pas la documentation vivante du repo
- le repo Git versionne reste la source de verite vivante
- une spec validee se ferme en mettant a jour la documentation vivante du repo
- les fichiers Word ne sont modifies que lors d'une tache documentaire explicite
